PROJECT 01 · FEATURED · OPEN SOURCE

SentinelBench — Detection Quality Benchmarking for Microsoft Sentinel

End-to-end detection quality benchmarking tool for Microsoft Sentinel — an open-source alternative to commercial platforms like AttackIQ. Executes 15 MITRE ATT&CK technique simulations via Atomic Red Team and validates whether Sentinel alerts, how fast, and at what severity.

Engineered a polling engine that queries the Log Analytics REST API at four checkpoints (T+2, T+5, T+10, T+15min) per technique, measures detection latency and severity delta, and auto-generates KQL detection rules for every missed detection. Delivered a React + D3.js dashboard with an ATT&CK-mapped latency heatmap, run history, and remediation panel backed by FastAPI + SQLite.
